# PYTHOn-CODeWar

Python practice solutions for Codewars kata.

## Solutions

- `even_or_odd(number: int) -> str`  
  Returns `"Even"` for even numbers, otherwise `"Odd"`.

- `number_to_string(num: int) -> str`  
  Converts an integer to a string.

- `no_space(x: str) -> str`  
  Removes all spaces from a string.

- `get_count(input_str: str) -> int`  
  Counts lowercase vowels (`a, e, i, o, u`) in a string.

## Run locally

```bash
python3 codewars_solutions.py
```

python3 -m pip install -U pytest
